# Invoke-ProfileBanner

## SYNOPSIS
Show my PowerShell Profile banner

## SYNTAX

```
Invoke-ProfileBanner [<CommonParameters>]
```

## DESCRIPTION
This function is purely used to write my PowerShell profile banner when I first open PowerShell

## EXAMPLES

### EXAMPLE 1
```
Invoke-ProfileBanner
```
