# Start-PowerShellAsSystem

## SYNOPSIS
Run PowerShell as system

## SYNTAX

```
Start-PowerShellAsSystem [<CommonParameters>]
```

## DESCRIPTION
This function uses PsExec to start a new PowerShell console as the system account

## EXAMPLES

### EXAMPLE 1
```
Start-PowerShellAsSystem
```

### EXAMPLE 2
```
SYS
```
