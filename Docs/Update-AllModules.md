# Update-AllModules

## SYNOPSIS
Update all PowerShell modules

## SYNTAX

```
Update-AllModules
```

## DESCRIPTION
This function runs all of the Node PowerShell module update commands one after the other to ensure all modules are updated

### EXAMPLE 1
